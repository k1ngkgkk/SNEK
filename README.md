# Snek

A mouse-controlled snake game built with Python and Pygame, featuring powerups, increasing difficulty, and tactical body management.

## Gameplay

* Move the mouse to guide the snake.
* Eat gray rats to grow and increase your score.
* Every 10 rats eaten increases rat speed.
* Collect the yellow wildcard buff to invert your mouse controls.
* Collect the blue Rush buff for a temporary speed boost.
* Collect the purple Calm buff to temporarily slow all rats down.
* Press **Space** to shed part of your body and leave behind a dead skin.
* Shed skins remain on the map and can destroy incoming rats.
* Prevent rats from colliding with your body.
* Press any key to restart after dying.

## Controls

| Action              | Control |
| ------------------- | ------- |
| Move Snake          | Mouse   |
| Shed Skin           | Space   |
| Restart After Death | Any Key |

## Setup

1. Install Pygame:

```bash
pip install pygame
```

2. Run the game:

```bash
python snek.py
```
